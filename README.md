# MyNotes
### Save your notes with image support and background colour
#### This is a simple note taking app with material design, meant for taking down a quick note.

**Core Concept**
* Room Database
* MVVM Model
    * Repository
    * ViewModel
    * Coroutines
* LiveData
* Fetching Image From Galary
* SearchView Implementation
